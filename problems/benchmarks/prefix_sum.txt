Title: Range Sum Queries

Statement:
You are given an array of N integers a[1..N] and Q queries. Each query consists of two integers L and R (1 <= L <= R <= N). For each query, print the sum of elements from index L to R inclusive, i.e., a[L] + a[L+1] + ... + a[R].

Constraints:
1 <= N <= 10^5
1 <= Q <= 10^5
-10^9 <= a[i] <= 10^9
Time limit: 2.0 seconds
Memory limit: 256 MB

Input format:
Line 1: two integers N and Q
Line 2: N space-separated integers a[1..N]
Next Q lines: each line contains two integers L and R

Output format:
Q lines, each containing the answer to the respective query.

Sample Input 1:
5 3
1 -2 3 -4 5
1 3
2 4
1 5

Sample Output 1:
2
-3
3
