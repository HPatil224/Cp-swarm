Title: Allocation of Books

Statement:
You are given N books, where the i-th book has a[i] pages. You need to allocate all these books to M students such that:
1. Each book is allocated to exactly one student.
2. Contiguous books must be allocated to the same student (i.e., a student gets a contiguous subarray of books).
3. The maximum number of pages allocated to any student is minimized.
Find this minimized maximum value. If it is impossible to allocate (e.g. N < M), print -1.

Constraints:
1 <= N <= 10^5
1 <= M <= N
1 <= a[i] <= 10^5
Time limit: 2.0 seconds
Memory limit: 256 MB

Input format:
Line 1: two integers N and M
Line 2: N space-separated integers a[1..N]

Output format:
A single integer: the minimized maximum sum of pages.

Sample Input 1:
4 2
12 34 67 90

Sample Output 1:
113

Sample explanation 1:
Book allocations:
Student 1 gets books 1, 2, 3 (12+34+67 = 113 pages).
Student 2 gets book 4 (90 pages).
Maximum pages allocated to a student is max(113, 90) = 113. Any other partition yields a larger maximum.
